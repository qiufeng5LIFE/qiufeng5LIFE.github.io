spring-boot和vue前后端分离项目整合部署

vue和spring-boot的本地开发调试

更改vue项目config文件下的index.js文件
更改以下三项

  • assetsPublicPath：/后端项目名

  • host：localhost

  • port：8080 这个端口号与spring-boot设置启动的端口号不同就可以
